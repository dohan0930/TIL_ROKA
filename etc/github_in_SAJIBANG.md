# 사지방에서 github 들어가보기 ~~어디까지 안되나 해보자~~

## 일단 nslookup을 해봅시다.
일단 저는 사지방 DNS 서버를 1.1.1.1로 바꿨습니다.
C:\Windows\System32의 CMD를 찾아서  

    netsh interface ip set dns "로컬 영역 연결" static 1.1.1.1
    
한 상태.

github의 IP는 192.30.255.113 / 192.30.255.112로 간다고 나옵니다.
